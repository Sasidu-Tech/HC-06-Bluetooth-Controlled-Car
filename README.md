# HC-06 Bluetooth Controlled Car

## 📌 Overview
A Bluetooth-controlled robot car built using Arduino UNO, HC-06 Bluetooth module, L293D motor driver, and DC motors.

## 🔧 Components
- Arduino UNO
- HC-06 Bluetooth Module
- L293D Motor Driver
- 2 DC Motors
- Robot Chassis
- Battery Pack

## 📱 Controls
- F - Forward
- B - Backward
- L - Left
- R - Right
- S - Stop

## 📂 Project Files
- Arduino source code
- Circuit diagram
- Project images
- Testing Video
